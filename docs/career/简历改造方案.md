## 四、简历项目重新包装方案

### 4.1 项目一：基于 RAG 与 Function Calling 的智能文档任务 Agent（核心项目）

**定位**：展示 AI Agent 系统设计、RAG、工具编排、质量评估综合能力。

**项目标题**：
> 基于 RAG 与 Function Calling 的智能文档任务 Agent

**项目简介**：
> 面向 MOD 开发场景（或 Python 学习场景）的智能 Agent 系统，整合 RAG 知识检索与 Function Calling 工具调用能力，支持文档检索、代码执行、文件读取等多工具协同，并内置回答质量评估模块，形成"生成-评估-优化"闭环。

**负责模块**：
- **RAG 检索增强模块**：基于 LangChain 实现文档加载、文本分块、Embedding 向量化与 Chroma 向量检索，封装为 Agent 可调用的 `search_docs` 工具
- **Function Calling 工具编排**：设计并实现真实工具（文档检索、Python 代码执行、文件读取），完成 Agent 决策 → 工具调用 → 结果回传的完整循环
- **Agent 核心循环**：实现多轮对话管理、工具调用日志、最大循环次数保护、异常处理与错误恢复
- **回答质量评估**：从正确性、相关性、完整性、安全性等维度对 Agent 输出进行自动评估，支持规则校验与 LLM-as-a-Judge

**技术亮点**：
- 深入理解 RAG 架构，掌握文本分块策略、Embedding 模型选型与向量检索优化
- 掌握 Function Calling 机制，能将自然语言指令转化为结构化工具调用
- 具备 Agent 系统设计能力，理解工具编排、多轮对话、上下文管理、错误恢复
- 建立回答质量评估闭环，提升 Agent 输出的可靠性

**技术栈**：Python、LangChain、FastAPI、Chroma、DashScope（通义千问）、Embedding 模型
